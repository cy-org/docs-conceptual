# Character &#39;character&#39; (0x&lt;hexadecimal&gt;) is not allowed in XML literal
An unsupported character has been used in an XML literal.  
  
 **Error ID:** BC31176  
  
## To correct this error  
  
-   Remove the unsupported character.
