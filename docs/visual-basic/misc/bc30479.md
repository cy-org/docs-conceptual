# Shared &#39;Sub New&#39; cannot have any parameters
You have included a parameter or parameters in a shared `Sub New` statement, which is not valid.  
  
 **Error ID:** BC30479  
  
## To correct this error  
  
-   Remove the parameter or parameters.
