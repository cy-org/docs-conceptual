# &#39;As&#39; expected
A declaration occurs that does not supply an `As` clause for one or more of the elements being declared.  
  
 **Error ID:** BC30197  
  
## To correct this error  
  
-   Furnish the `As` clause as required, or turn `Option Strict Off`.
