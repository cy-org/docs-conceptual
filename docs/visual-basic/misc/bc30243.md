# &#39;&lt;modifier&gt;&#39; is not valid on an event declaration
An `Event` statement contains an invalid keyword, such as `ReadOnly`.  
  
 **Error ID:** BC30243  
  
## To correct this error  
  
1.  Remove the invalid keyword from the `Event` statement.
