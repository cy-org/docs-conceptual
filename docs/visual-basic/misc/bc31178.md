# Expected closing &#39;;&#39; for XML entity
An XML entity reference (for example, `>`) is missing a closing semicolon (;).  
  
 **Error ID:** BC31178  
  
## To correct this error  
  
-   Include the semicolon at the end of the XML entity reference.
