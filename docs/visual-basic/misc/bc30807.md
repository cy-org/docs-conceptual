# &#39;Let&#39; and &#39;Set&#39; assignment statements are no longer supported
`Let` and `Set` cannot be used in assignments.  
  
 **Error ID:** BC30807  
  
## To correct this error  
  
-   Omit `Let` and `Set` from assignment statements; for example:  
  
     `X = 2`
