# Argument not specified for parameter &#39;&lt;parametername&gt;&#39; of extension method &#39;&lt;methodname&gt;&#39; defined in &#39;&lt;typename&gt;&#39;
An argument for a required parameter in an extension method has not been supplied.  
  
 **Error ID:** BC36586  
  
## To correct this error  
  
-   Supply an argument for the specified parameter.
