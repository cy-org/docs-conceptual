# Compiler Warning (level 1) CS2029
Invalid value for '/define'; 'identifier' is not a valid identifier  
  
 This warning occurs if the value that is used in the [/define](../../csharp/language-reference/compiler-options/define-compiler-option.md) option has some invalid characters.  
  
 This warning cannot be suppressed by the [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) option.
