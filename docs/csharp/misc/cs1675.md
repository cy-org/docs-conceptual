# Compiler Error CS1675
Enums cannot have type parameters  
  
 To resolve this error, remove the type parameter from the `enum` declaration.  
  
## Example  
 The following sample generates CS1675:  
  
```  
// CS1675.cs  
enum E<T>  // CS1675  
{  
}  
  
class CMain  
{  
    public static void Main()  
    {  
    }  
}  
```
