# Compiler Error CS0177
The out parameter 'parameter' must be assigned to before control leaves the current method  
  
 A parameter marked with the [out](../../csharp/language-reference/keywords/out.md) keyword was not assigned a value in the method body. For more information, see [Passing Parameters](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)  
  
 The following sample generates CS0177:  
  
```  
// CS0177.cs  
public class MyClass  
{  
   public static void Foo(out int i)   // CS0177  
   {  
   // uncomment the following line to resolve this error  
   //   i = 0;  
   }  
  
   public static void Main()  
   {  
       int x = -1;  
       Foo(out x);  
   }  
}  
```
