# Compiler Error CS1558
'class' does not have a suitable static Main method  
  
 The [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method. However, the [Main](../../csharp/programming-guide/main-and-command-args/index.md) method was not defined correctly.  
  
 The following example generates CS1558 because of invalid return type.  
  
```  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
