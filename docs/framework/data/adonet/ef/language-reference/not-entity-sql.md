# ! (NOT) (Entity SQL)
Negates a `Boolean` expression.  
  
## Syntax  
  
```  
NOT boolean_expression  
or  
! boolean_expression  
```  
  
## Arguments  
 `boolean_expression`  
 Any valid expression that returns a Boolean.  
  
## Remarks  
 The exclamation point (!) has the same functionality as the NOT operator.
