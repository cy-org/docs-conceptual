# ProtocolStopped
Id: 151  
  
 Severity: Error  
  
 Category: TransactionBridge  
  
## Description  
 This event indicates that the WS-Atomic Transaction protocol service was stopped. The event lists the protocol ID, protocol name, process name and process ID. .
