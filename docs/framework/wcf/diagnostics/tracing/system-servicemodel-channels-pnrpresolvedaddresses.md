# System.ServiceModel.Channels.PnrpResolvedAddresses
The Pnrp Resolver completed a resolve operation.  
  
## Description  
 This warning trace indicates that the Pnrp Resolver could not find any addresses. This trace occurs during a Resolve Operation by the PNRP Resolver.
