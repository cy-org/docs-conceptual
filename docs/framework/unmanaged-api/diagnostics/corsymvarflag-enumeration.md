# CorSymVarFlag Enumeration
Indicates whether a variable is compiler-generated.  
  
## Syntax  
  
```  
typedef enum CorSymVarFlag   
{  
    VAR_IS_COMP_GEN = 1  
} CorSymVarFlag;  
```  
  
## Members  
  
|Member|Description|  
|------------|-----------------|  
|`VAR_IS_COMP_GEN`|Indicates that the given variable is compiler-generated.|  
  
## Requirements  
 **Header:** CorSym.idl, CorSym.h
