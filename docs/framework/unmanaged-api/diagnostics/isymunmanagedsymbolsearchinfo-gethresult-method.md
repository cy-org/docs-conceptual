# ISymUnmanagedSymbolSearchInfo::GetHRESULT Method
Gets the HRESULT.  
  
## Syntax  
  
```  
HRESULT GetHRESULT(  
    [out] HRESULT *phr);  
```  
  
#### Parameters  
 `phr`  
 [out] A pointer to the HRESULT.  
  
## Return Value  
 S_OK if the method succeeds; otherwise, E_FAIL or some other error code.  
  
## Requirements  
 **Header:** CorSym.idl, CorSym.h
