# ISymUnmanagedWriter3::Commit Method
Commits the changes written so far to the stream.  
  
## Syntax  
  
```  
HRESULT Commit();  
```  
  
## Return Value  
 S_OK if the method succeeds; otherwise, E_FAIL or some other error code.  
  
## Requirements  
 **Header:** CorSym.idl , CorSym.h
