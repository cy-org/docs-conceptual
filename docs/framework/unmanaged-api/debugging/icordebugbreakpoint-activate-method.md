# ICorDebugBreakpoint::Activate Method
Sets the active state of this `ICorDebugBreakpoint`.  
  
## Syntax  
  
```  
HRESULT Activate (  
    [in] BOOL bActive  
);  
```  
  
#### Parameters  
 `bActive`  
 [in] Set this value to `true` to specify the state as active; otherwise, set this value to `false`.
