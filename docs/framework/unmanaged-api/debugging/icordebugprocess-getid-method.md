# ICorDebugProcess::GetID Method
Gets the operating system (OS) ID of the process.  
  
## Syntax  
  
```  
HRESULT GetID([out] DWORD *pdwProcessId);  
```  
  
#### Parameters  
 `pdwProcessId`  
 [out] The unique ID of the process.
