# ICorDebugInstanceFieldSymbol::GetSize Method
Gets the size in bytes of the instance field.  
  
## Syntax  
  
```  
HRESULT GetSize(  
   [out] ULONG32 *pcbSize  
);  
```  
  
#### Parameters  
 `pcbSize`  
 [out] A pointer to length of the field.  
  
## Remarks  
  
> [!NOTE]
>  This method is available with .NET Native only.
