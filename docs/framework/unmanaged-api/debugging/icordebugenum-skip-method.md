# ICorDebugEnum::Skip Method
Moves the cursor forward in the enumeration by the specified number of items.  
  
## Syntax  
  
```  
HRESULT Skip (  
    [in] ULONG celt  
);  
```  
  
#### Parameters  
 `celt`  
 [in] The number of items by which to move the cursor forward.
