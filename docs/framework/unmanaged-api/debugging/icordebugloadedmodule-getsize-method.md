# ICorDebugLoadedModule::GetSize Method
Gets the size in bytes of the loaded module.  
  
## Syntax  
  
```  
HRESULT GetSize(  
   [out] ULONG32 *pcBytes  
);  
```  
  
#### Parameters  
 `pcBytes`  
 [out] A pointer to the number of bytes in the loaded module.  
  
## Remarks  
  
> [!NOTE]
>  This method is available with .NET Native only.
