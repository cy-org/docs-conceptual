# ICorDebugFunctionBreakpoint::GetOffset Method
Gets the offset of the breakpoint within the function.  
  
## Syntax  
  
```  
HRESULT GetOffset (  
    [out] ULONG32  *pnOffset  
);  
```  
  
#### Parameters  
 `pnOffset`  
 [out] A pointer to the offset of the breakpoint.
