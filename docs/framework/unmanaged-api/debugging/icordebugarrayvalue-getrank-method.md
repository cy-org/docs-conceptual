# ICorDebugArrayValue::GetRank Method
Gets the number of dimensions in the array.  
  
## Syntax  
  
```  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
#### Parameters  
 `pnRank`  
 [out] A pointer to the number of dimensions in this `ICorDebugArrayValue` object.
