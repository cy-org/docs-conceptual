# ICorDebugMemoryBuffer::GetSize Method
Gets the size of the memory buffer in bytes.  
  
## Syntax  
  
```  
HRESULT GetSize(  
   [out] ULONG32 *pcbBufferLength  
);  
```  
  
#### Parameters  
 `pcbBufferLength`  
 [out] A pointer to the size of the memory buffer.  
  
## Remarks  
  
> [!NOTE]
>  This method is available with .NET Native only.
