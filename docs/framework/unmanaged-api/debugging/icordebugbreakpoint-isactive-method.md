# ICorDebugBreakpoint::IsActive Method
Gets a value that indicates whether this `ICorDebugBreakpoint` is active.  
  
## Syntax  
  
```  
HRESULT IsActive (  
    [out] BOOL *pbActive  
);  
```  
  
#### Parameters  
 `pbActive`  
 [out] `true` if this breakpoint is active; otherwise, `false`.
