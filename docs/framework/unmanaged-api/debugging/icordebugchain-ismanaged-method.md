# ICorDebugChain::IsManaged Method
Gets a value that indicates whether this chain is running managed code.  
  
## Syntax  
  
```  
HRESULT IsManaged (  
    [out] BOOL               *pManaged  
);  
```  
  
#### Parameters  
 `pManaged`  
 [out] `true` if this chain is running managed code; otherwise, `false`.
