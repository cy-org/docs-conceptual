# ICorDebugBoxValue::GetObject Method
Gets the boxed value.  
  
## Syntax  
  
```  
HRESULT GetObject (  
    [out] ICorDebugObjectValue **ppObject  
);  
```  
  
#### Parameters  
 `ppObject`  
 [out] A pointer to the address of an ICorDebugObjectValue object that represents the boxed value.
