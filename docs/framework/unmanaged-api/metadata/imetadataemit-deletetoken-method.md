# IMetaDataEmit::DeleteToken Method
Deletes the specified token from the current metadata scope.  
  
## Syntax  
  
```  
HRESULT DeleteToken (   
    [in]  mdToken     tkObj   
);  
```  
  
#### Parameters  
 `tkObj`  
 [in] The token to be deleted.
