# IMetaDataEmit2::ResetENCLog Method
Resets the edit-and-continue log and starts a new session.  
  
## Syntax  
  
```  
HRESULT ResetENCLog ();  
```  
  
#### Parameters  
 `ResetENCLog` takes no parameters.
