# IMetaDataAssemblyImport::CloseEnum Method
Releases a reference to the specified enumeration instance.  
  
## Syntax  
  
```  
void CloseEnum (  
    [in] HCORENUM     hEnum  
);  
```  
  
#### Parameters  
 `hEnum`  
 [in] The enumeration instance to be closed.
