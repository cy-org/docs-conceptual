# IMetaDataTables::GetStringHeapSize Method
Gets the size, in bytes, of the string heap.  
  
## Syntax  
  
```  
HRESULT GetStringHeapSize (  
    [out] ULONG   *pcbStrings  
);  
```  
  
#### Parameters  
 `pcbStrings`  
 [out] A pointer to the size, in bytes, of the string heap.
