# ICeeGen::GenerateCeeMemoryImage Method
Generates an image in memory for the code base.  
  
 This method is obsolete and should not be used.  
  
## Syntax  
  
```  
HRESULT GenerateCeeMemoryImage (  
    [out] void    **ppImage  
);  
```  
  
#### Parameters  
 `ppImage`  
 [out] A pointer to the generated image.
