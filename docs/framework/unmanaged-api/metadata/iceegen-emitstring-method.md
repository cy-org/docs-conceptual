# ICeeGen::EmitString Method
Emits the specified string into the code base.  
  
 This method is obsolete and should not be used.  
  
## Syntax  
  
```  
HRESULT EmitString (  
    [in]  LPWSTR    lpString,  
    [out] ULONG     *RVA  
);  
```  
  
#### Parameters  
 `lpString`  
 [in] The string to emit.  
  
 `RVA`  
 [out] The relative virtual address of the emitted string.
