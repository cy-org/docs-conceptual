# CVStruct Structure
Contains information that is used when installing a module or a composite image.  
  
## Syntax  
  
```  
typedef struct {  
    short Major;  
    short Minor;  
    short Sub;  
    short Build;  
} CVStruct;  
```  
  
## Members  
  
|Member|Description|  
|------------|-----------------|  
|Major|Major version build number.|  
|Minor|Minor version build number.|  
|Sub|Sub-build number.|  
|Build|Build number.|
