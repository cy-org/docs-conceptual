# IMetaDataEmit::SaveToStream Method
Saves all metadata in the current scope to the specified `IStream`.  
  
## Syntax  
  
```  
HRESULT SaveToStream (   
    [in]  IStream     *pIStream,  
    [in]  DWORD       dwSaveFlags  
);  
```  
  
#### Parameters  
 `pIStream`  
 [in] The writable stream to save to.  
  
 `dwSaveFlags`  
 [in] Reserved. Must be zero.
