# IMetaDataEmit2::SaveDelta Method
Saves changes from the current edit-and-continue session to the specified file.  
  
## Syntax  
  
```  
HRESULT SaveDelta (  
    [in] LPCWSTR     szFile,   
    [in] DWORD       dwSaveFlags  
);  
```  
  
#### Parameters  
 `szFile`  
 [in] The file name under which to save changes.  
  
 `dwSaveFlags`  
 [in] Reserved. Must be zero.
