# IMetaDataTables::GetGuidHeapSize Method
Gets the size, in bytes, of the GUID heap.  
  
## Syntax  
  
```  
HRESULT GetGuidHeapSize (  
    [out] ULONG   *pcbGuids  
);  
```  
  
#### Parameters  
 `pcbGuids`  
 [out] A pointer to the size, in bytes, of the GUID heap.
