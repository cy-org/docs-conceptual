# ICLRDebugManager::GetDacl Method
This method is not implemented.  
  
## Syntax  
  
```  
HRESULT GetDacl (  
    [out] PACL* ppacl  
);  
```  
  
#### Parameters  
 `ppacl`  
 [out] An interface pointer to the Access Control List (ACL).  
  
## Return Value  
  
|HRESULT|Description|  
|-------------|-----------------|  
|E_NOTIMPL|The method is not implemented.|
