# ICLRDebugManager::SetDacl Method
This method is not implemented.  
  
## Syntax  
  
```  
HRESULT SetDacl (  
    [in] PACL pacl  
);  
```  
  
#### Parameters  
 `pacl`  
 [in] A pointer to the Access Control List (ACL).  
  
## Return Value  
  
|HRESULT|Description|  
|-------------|-----------------|  
|E_NOTIMPL|The method is not implemented.|
