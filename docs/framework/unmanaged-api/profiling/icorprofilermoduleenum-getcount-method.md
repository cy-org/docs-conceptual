# ICorProfilerModuleEnum::GetCount Method
Gets the number of managed modules that were loaded into the application.  
  
## Syntax  
  
```  
HRESULT GetCount([out] ULONG * pcelt);  
```  
  
#### Parameters  
 `celt`  
 [out] The number of runtime modules in the collection.
