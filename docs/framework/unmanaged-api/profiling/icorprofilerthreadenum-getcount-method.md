# ICorProfilerThreadEnum::GetCount Method
Gets the number of threads that are used by the application.  
  
## Syntax  
  
```  
HRESULT GetCount (    [out] ULONG * pcelt  
);  
```  
  
#### Parameters  
 `celt`  
 [out] The number of threads used by the application.
